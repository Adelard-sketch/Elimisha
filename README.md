# Elimisha Education Center Website

A modern, responsive website for Elimisha Education Center - an educational institution dedicated to empowering minds and transforming lives through quality education.

## About

Elimisha Education Center (meaning "to educate" in Swahili) is a community-focused educational institution founded in 2010. The center offers holistic programs addressing academic excellence, emotional growth, and practical life skills for learners of all ages.

## Features

- **Responsive Design** - Fully responsive layout that works on all devices
- **Modern UI/UX** - Clean, contemporary design with smooth animations
- **Interactive Navigation** - Sticky navigation with hamburger menu for mobile
- **Scroll Animations** - Elements reveal on scroll for engaging user experience
- **Program Showcase** - Detailed program cards with images and descriptions
- **Statistics Section** - Key metrics highlighting the center's impact
- **Testimonials** - Community voices and success stories
- **Team Section** - Meet the educators and leadership
- **Contact Form** - Easy-to-use contact form for inquiries
